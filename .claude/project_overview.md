# Обзор проекта: Нонограмм

## Что это

Мобильная игра-головоломка **Нонограмм** (Picross) на Flutter.
Игрок решает логические задачи: заполняет клетки сетки по числовым подсказкам, расположенным по краям поля.

## Цели продукта

- Предоставить коллекцию Нонограмм-уровней с нарастающей сложностью.
- Сохранять прогресс игрока локально.
- Обеспечить простой и понятный интерфейс без лишних отвлечений.
- Быть расширяемым: легко добавлять новые уровни через JSON-файлы.

## Аудитория

Игроки, любящие логические пазлы. Приложение работает офлайн.

## MVP

1. Экран выбора сложности (лёгкий / сложный).
2. Экран списка уровней с отметкой пройденных.
3. Игровой экран с полным управлением.
4. Сохранение прогресса в локальную БД.

## Технологии

Flutter + Riverpod + Drift + GoRouter + Freezed.
Подробнее — в `tech_stack.md`.
