# Механика игры Нонограмм

## Правила

Нонограмм (также Picross, Griddler) — логическая головоломка на сетке.

Цель: заполнить ячейки сетки так, чтобы выполнялись все числовые подсказки по строкам и столбцам.

### Подсказки

Подсказки — числа, расположенные слева от каждой строки и сверху каждого столбца.

Каждое число означает **длину непрерывной группы** закрашенных клеток в этой строке/столбце.

Пример подсказки `[3, 1]` означает: сначала идёт группа из 3 закрашенных клеток, затем — как минимум одна пустая, затем — группа из 1 клетки.

Подсказка `[0]` означает, что в строке/столбце нет ни одной закрашенной клетки.

### Победное условие

Игрок побеждает, когда все строки и столбцы `playerGrid` полностью соответствуют подсказкам. В реализации это проверяется сравнением: каждая клетка с `CellState.filled` должна совпадать с `solution[row][col] == 1`.

### Режимы

- **Fill** — нажатие закрашивает пустую клетку или убирает закрашенную.
- **Cross** — нажатие ставит крест или убирает его. Кресты помогают отмечать клетки, которые точно пустые.

### Генерация подсказок

Для строки: сканируем слева направо, подсчитываем длины непрерывных групп единиц в `solution`.

Для столбца: то же самое, только сканируем сверху вниз.

Пример:
```
Строка: [0, 1, 1, 0, 1]
Подсказка: [2, 1]
```

### Ограничения MVP

- Нет таймера.
- Нет подсказок игроку при ошибке.
- Нет отмены хода (только полный сброс поля).
